# `docs/presentation/`

Use this folder to prepare the final slide deck.

## Recommended slide flow

1. Problem and motivation
2. Why the original scope was changed
3. Dataset and scale
4. Representations compared
5. Evaluation tasks
6. Result summary table
7. Latent-space insights
8. Error analysis
9. Final conclusion

## Rule

Every slide should answer one question clearly. No vague slides.
