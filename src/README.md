# `src/`

Reusable project code lives here.

## Subfolders

- `data/`: loading, cleaning, sampling, splitting
- `features/`: TF-IDF, LSA, document embeddings
- `models/`: training and probe models
- `evaluation/`: metrics, analysis, latent-space tools
- `utils/`: helpers used across modules

## Rule

A notebook should never be the only place where critical logic exists.
