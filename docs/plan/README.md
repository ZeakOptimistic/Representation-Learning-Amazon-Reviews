# `docs/plan/`

This is the control center for project management.

## Files

- `master_plan.md`: full project blueprint
- `milestones.md`: week-by-week execution plan
- `risk_register.md`: main project risks and mitigations
- `decision_log.md`: scope decisions and why they were made

## Rule

No major scope changes without an entry in `decision_log.md`.
